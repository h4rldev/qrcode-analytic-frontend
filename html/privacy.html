Privacy Policy

As the operator of this service, I prioritize the privacy and security of user information. This Privacy Policy outlines the practices concerning the collection and usage of data, specifically IP addresses that are logged for the sole purpose of maintaining the security and integrity of this service.

IP addresses are collected as part of the server logging process when users interact with this service. This information is crucial for ensuring the security, stability, and performance of the service.

The collected IP addresses are used solely for maintaining the security and integrity of this service. Server logs are analyzed to identify and prevent malicious activities, such as hacking attempts, fraud, and other unauthorized access.

Industry-standard security measures are employed to protect the confidentiality and integrity of the collected IP addresses. This includes encryption, access controls, and regular monitoring of server logs to detect and mitigate security threats.

Server logs containing IP addresses are retained for a limited period, as necessary to fulfill the purposes outlined in this Privacy Policy and to comply with legal requirements. After this period, logs are securely deleted or anonymized.
